Swish-e Daily Development Snapshots

The following Swish-e packages are generated daily from Subversion checkout.

These are not official releases of Swish-e, rather the current development version. As such, there is no guarantee that these packages run. Please do not use this code in production.

These are source snapshots. See the Swish-e [% link_to_page('download') %] page for released versions of Swish-e.

For Windows development binary (pre-compiled) snapshots, please see /http://www.webaugur.com/wares/files/swish-e/daily/. The most current Windows development version can be found at here.

Questions regarding these files, or about using Swish-e in general, should be directed to the Swish-e mailing list. See the [% link_to_page('discuss') %] page for more information.
