Downloads

Swish-e version [% swish.current_version %] is the latest stable release. Unless you are a Windows user, you should download the latest source version. See the included [% link_to_page('docs') %] for instructions on [% link_to_page( 'install', 'installing' ) %] and using Swish-e. Questions should go to the [% link_to_page( 'discuss', 'Swish-e discussion list' ) %].

Daily snapshots of the development source code can be found on the [% link_to_page( 'daily' ) %] page.

Other files on this download page are archives of previous versions.

Windows Note

The Windows version of Swish-e contains the swish-e binary program and required libraries (libxml2, iconv and zlib) and also catdoc and xpdf for parsing MS Word and PDF files.
